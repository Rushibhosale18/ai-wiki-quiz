## AI Wiki Quiz Generator

### Tech Stack
- FastAPI
- PostgreSQL
- LangChain
- Gemini API
- BeautifulSoup
- HTML

### Features
- Generate quizzes from Wikipedia
- AI-based question generation
- Quiz history
- JSON API

### How to Run
1. Set GEMINI_API_KEY
2. Run backend:
   uvicorn main:app --reload
3. Open frontend/index.html

### Endpoints
- POST /generate-quiz
- GET /history
